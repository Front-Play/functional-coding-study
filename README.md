## ⚙️ 진행방식

- 1인당 1챕터를 읽고 정리하여 발표.
- 매주 2챕터 진행 (각자 1챕터 필수)

<br/>

### commit 컨벤션

```
  docs: md파일 작성
  edit: 내용 수정
  del: 파일 혹은 폴더 삭제
  rename: 파일명 수정
```

### 파일 형식

- 마크다운 파일로 작성(.md)
- 파일은 해당 챕터 폴더에 커밋
  - 파일명은 **_챕터넘버\_챕터명_** 으로 작성
    - ex) `01_타입스크립트`
  - 공백은 `_`로 작성

### PR제목

```
  [작성자명] 작성한 내용 요약
```

- ex) `[홍길동] OOO의 종류와 이해`
- PR의 내용은 자율적으로 작성
